# Code de conduite

Ce projet accueille toute personne, quel que soit son parcours. En
participant (issues, PR, discussions), vous acceptez de :

- rester respectueux et factuel, critiquer le code et les idées, jamais les
  personnes ;
- ne pas harceler, discriminer, ni publier d'informations privées d'autrui ;
- accepter les décisions de mainteneurs motivées par les règles du projet
  (notamment les règles non négociables de CONTRIBUTING.md) ;
- signaler les comportements problématiques au mainteneur en privé.

Les manquements peuvent entraîner un avertissement, la suppression de
contributions, ou un bannissement du projet selon la gravité.
